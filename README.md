NayePankh Foundation — Official Website

A fully responsive, modern website built for NayePankh Foundation 
as part of a Front End Development Internship project.

✨ Features
- 🌙 Dark / Light mode toggle with localStorage persistence
- 🎞️ Scroll-triggered reveal animations
- 🔢 Animated impact counters (2L+ lives, 100+ volunteers, 50+ campaigns)
- 📱 Mobile-first responsive design with hamburger menu
- 🖼️ Interactive gallery with hover overlays
- 🃏 3D tilt effect on mission cards
- 🏷️ Sticky header with active section highlighting
- 📧 Email subscription form with validation
- ⬆️ Back-to-top button
- ♿ Reduced motion support for accessibility

🛠️ Built With
- HTML5
- CSS3 (custom properties, grid, flexbox, keyframe animations)
- Vanilla JavaScript (IntersectionObserver, requestAnimationFrame)
- Google Fonts (Playfair Display + Inter)

📁 Structure
├── index.html   → Page structure & content
├── style.css    → All styling, theming & animations
└── script.js    → Interactivity, dark mode, counters

🚀 Getting Started
Just open index.html in any browser — no build tools or dependencies needed.
